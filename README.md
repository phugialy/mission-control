# Mission Control UI

AI Agent Orchestration Platform - Control UI

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Prisma (SQLite)
- OpenClaw Agent Framework

## Getting Started

```bash
npm install
npm run dev
```

## Environment

See `.env` for required environment variables.

## Features

- Mission/Objective management
- Kanban board interface
- Scribe logging integration
- Agent heartbeat scheduling
